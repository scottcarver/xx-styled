== Description ==

This plugin adds a new Block ("xx/styled") and Posttype ("styled") which allows the user to create reusable, themed areas. These are called "styled areas." 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/xx-styled` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
